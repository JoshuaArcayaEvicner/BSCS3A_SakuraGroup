Notification Components

Making use of the onclick event:

So far in this course, when the body of the web page is loaded, we have called the showNotification() function. We could still use other events to activate it instead. For example, we may show the warning only if the user clicks on a button.

Consider this:
Remove the body> element's onload="showNotification()" property. Insert the following HTML button> element anywhere on your web page:

"button onclick="showNotification('successNotification')">Show Success Notification"
"button onclick="showNotification('errorNotification')">Show Error Notification"
"button onclick="showNotification('infoNotification')">Show Info Notification"

Shows the example here below:

Success: Your action was successful.
Error: An error occurred while processing your request.
Info: This is an informational message.

Slider

Description: This is a slider that transitions from light pink to dark pink.

HTML Structure:
A input> element in the HTML structure with the type set to "range" and the ID of "slider" is present. This input element stands in for the slider element.

JavaScript Logic:
The slider element in the given JavaScript code enables users to choose a numeric value by dragging a thumb along a track. As the thumb advances along the
track, the slider's color changes from light to dark. The selected value, which is shown as a number, is represented by the thumb's position. Depending on
where the thumb is placed, the code determines and updates the thumb and track's color. The color gradient and the value that is displayed are dynamically
updated as the user moves the slider.
